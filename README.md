# Trendyol Rakip Analizi

Next.js + shadcn/ui ile hazırlanmış Trendyol mağaza linki bazlı rakip analiz arayüz prototipi.

## Komutlar

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Mevcut durum

- Tek input: Trendyol mağaza veya ürün URL'si
- Shadcn/ui tabanlı analiz dashboard'u
- API route üzerinden deterministik örnek rapor üretimi
- Canlı scraper/API katmanı için genişletilebilir veri modeli

## Sonraki adımlar
